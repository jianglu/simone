功能性（competent）     : 对玩家有帮助
趣味性（Interesting）   : 有自己的性格
剧情性（story value）   : 推动剧情

第一步：为 NPC 创造一个性格（Personality）
第二步：为 NPC 创造一个情绪指数（MoodScale）：1-10评分
第三步：为每一个情绪指数设定一个反应
第四步：根据性格为 NPC 设定不同的反应
第五步：根据玩家的行为，设定 NPC 的心情变化
